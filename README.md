## 💌 এই ফাইলটা কী?

এটা কোনো সাধারণ project না।

এটা একজন বিশেষ মানুষের জন্য তৈরি করা একটা ডিজিটাল চিঠি — যেখানে কিছু কথা আছে যেগুলো সরাসরি বলা হয়নি, কিন্তু মনে অনেকদিন ধরে জমা ছিল। অনেকটা সময়, অনেকটা যত্ন আর একটু ভালোবাসা দিয়ে বানানো।

যে এটা পাবে — সে জানুক, এই চিঠিটা শুধু তার জন্যই।

## 📥 কীভাবে ডাউনলোড করবে?

> **ফাইলের নাম:** `surprise.html`
> **ফাইলটা কোথায়:** এই README-র ঠিক নিচেই দেখতে পাবে

<br/>

নিচের ধাপগুলো অনুসরণ করো 👇

<br/>

### ধাপ ১ — ফাইলটা খোলো

এই পেজের উপরের দিকে তাকাও — `README.md`-এর ঠিক নিচে **`surprise.html`** লেখাটা দেখতে পাবে।
সেই লেখার উপর **ক্লিক** করো।

<br/>

```
📁 Repository
 ├── 📄 README.md      ← তুমি এখন এখানে আছ
 └── 🎀 surprise.html  ← এইটায় ক্লিক করো
```

<br/>

### ধাপ ২ — Download করো

`surprise.html`-এ ক্লিক করার পর একটা নতুন পেজ খুলবে।
সেই পেজের **উপরের ডান কোণে** তিনটা ডট ( **⋯** ) দেখতে পাবে — ওটায় চাপ দাও।

সেখানে **"Download"** অপশন আসবে — সেটায় চাপ দিলেই ফাইলটা তোমার ফোনে/কম্পিউটারে নেমে যাবে।

<br/>

> 💡 **মোবাইলে থাকলে:** ফাইলটা সাধারণত `Downloads` ফোল্ডারে সেভ হবে।

<br/>

### ধাপ ৩ — Chrome-এ খোলো

ফাইলটা ডাউনলোড হওয়ার পর —

- **ফোনে:** File Manager → Downloads → `surprise.html` → Click → **Google Chrome**
- **কম্পিউটারে:** ডাউনলোড হওয়া ফাইলটার উপর ডাবল ক্লিক করো, অথবা Chrome-এ drag করে ছেড়ে দাও

<br/>

> ⚠️ **অবশ্যই Google Chrome ব্যবহার করবে** — অন্য browser-এ ঠিকমতো কাজ নাও করতে পারে।

<br/>

### ধাপ ৪ — উপভোগ করো 🎉

Chrome-এ খোলার পর শুধু screen-এ **tap বা click** করতে থাকো — এক পাতা থেকে আরেক পাতায় যাবে।
**Sound on রেখো** — পুরো experience-টা অনেক বেশি ভালো লাগবে। 🔊
