# 🎂 Birthday Surprise — একটি ইন্টারঅ্যাক্টিভ জন্মদিনের সারপ্রাইজ ওয়েবপেজ

একজন প্রিয় মানুষের জন্মদিন উপলক্ষে তৈরি করা একটি সম্পূর্ণ ইন্টারঅ্যাক্টিভ, অ্যানিমেটেড ওয়েব এক্সপেরিয়েন্স। পুরো প্রজেক্টটি **মাত্র একটি HTML ফাইলে** তৈরি — কোনো সার্ভার, ইন্টারনেট কানেকশন বা আলাদা অ্যাসেট ফোল্ডার ছাড়াই সরাসরি ব্রাউজারে খুলে চালানো যায়।

---

## ✨ ফিচারসমূহ

- 🎬 **দৃশ্য-ভিত্তিক স্টোরিটেলিং** — একটার পর একটা দৃশ্য (scene) ধাপে ধাপে উন্মোচিত হয়, ট্যাপ করে এগিয়ে যাওয়া যায়
- 🎵 **এমবেডেড ব্যাকগ্রাউন্ড মিউজিক** — Base64 এনকোডেড অডিও, ফাইল খুললেই বাজতে শুরু করে এবং লুপ হয়ে চলতে থাকে
- 🌸 **কাস্টম অ্যানিমেশন** — ফ্লোটিং পাপড়ি, কনফেটি, ফায়ারওয়ার্কস ইফেক্ট, পার্টিকল সিস্টেম
- 💌 **ইন্টারঅ্যাক্টিভ এনভেলাপ ও চিঠি** — খাম খোলার অ্যানিমেশন থেকে শুরু করে পেজ-বাই-পেজ চিঠি পড়া
- 📱 **মোবাইল-ফ্রেন্ডলি** — ট্যাপ-টু-কন্টিনিউ নেভিগেশন, প্রোগ্রেস ডট, রেসপন্সিভ লেআউট
- 📦 **সিঙ্গেল-ফাইল আর্কিটেকচার** — সব ছবি ও মিউজিক একই `.html` ফাইলের ভেতরে Base64 আকারে এমবেডেড, তাই শুধু একটা ফাইল শেয়ার করলেই হয়

---

## 📂 ফাইল স্ট্রাকচার

```
.
└── surprise_combined.html   # সম্পূর্ণ সারপ্রাইজ এক্সপেরিয়েন্স — একটাই ফাইল
```

---

## ▶️ কীভাবে দেখবেন

1. **`surprise_combined.html`** ফাইলটি ডাউনলোড করুন
2. যেকোনো আপ-টু-ডেট ব্রাউজারে (**Chrome প্রস্তাবিত**) ফাইলটি ওপেন করুন
3. ফোনের সাউন্ড অন রেখে স্ক্রিনে ট্যাপ করে ধাপে ধাপে এগিয়ে যান

> 💡 মোবাইলে সবচেয়ে ভালো এক্সপেরিয়েন্স পাওয়া যায়। ফোনের ভলিউম ফুল রাখা এবং স্ক্রিন অটো-লক টাইম কিছুটা বাড়িয়ে রাখার পরামর্শ দেওয়া হচ্ছে।

## 🛠️ ব্যবহৃত প্রযুক্তি

| প্রযুক্তি | ব্যবহার |
|---|---|
| **HTML5** | স্ট্রাকচার ও সিনটিক মার্কআপ |
| **CSS3** | অ্যানিমেশন, ট্রানজিশন, ফ্লেক্সবক্স লেআউট |
| **Vanilla JavaScript** | দৃশ্য নিয়ন্ত্রণ, মিউজিক হ্যান্ডলিং, পার্টিকল/ফায়ারওয়ার্ক ইফেক্ট (কোনো ফ্রেমওয়ার্ক ছাড়াই) |
| **Base64 Encoding** | অডিও ও ইমেজ অ্যাসেট সরাসরি কোডের ভেতরে এমবেড করার জন্য |
